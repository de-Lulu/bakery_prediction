STRG+SHIFT+P -> öffnet Befehlspalette
STRG+` -> öffnet Terminal 
STRG + W -> schließt aktuellen Tab
git add <datei> -> um eine Datei zur Staging Area hinzuzufügen
git commit -m "Nachricht" -> um die Änderungen zu committen
git push origin main -> um die Änderungen ins Remote-Repository zu pushen
git pull origin main -> um die Änderungen vom Remote-Repository zu holen
git checkout <branch-name> -> um zu einem anderen Branch zu wechseln
git branch <branch-name> -> um einen neuen Branch zu erstellen
git merge <branch-name> -> um einen Branch in den aktuellen Branch zu mergen

cherry picken: in der Versionshistorie bestimmten Commit ansehen 
und in den aktuellen Branch übernehmen.
Dafür:
git log -> um die Commit-Historie anzusehen (oder einfach auf die Uhr klicken)
cherry-pick <commit-id> -> um einen bestimmten Commit zu übernehmen
